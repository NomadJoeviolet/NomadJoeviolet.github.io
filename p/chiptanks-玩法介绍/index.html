2025-11-27

🎮 游戏介绍：Chip Tanks (芯片坦克)

Chip Tanks  是一款运行在 STM32 嵌入式平台上的  Roguelike 俯视视角弹幕射击游戏。玩家将操控一辆微型坦克，在 OLED 屏幕呈现的战场上，迎战源自《山海经》与中国上古神话的机械化凶兽。

通过击败敌人获取经验，升级并抽取强力的  PerkCard（天赋卡），构建属于你的战斗流派，挑战强大的关卡 BOSS！

🕹️ 游戏类型与核心机制

  • 类型：Roguelike / 弹幕射击 (STG) / 嵌入式游戏
  • 核心循环：战斗 -> 积累经验 -> 升级 -> 三选一抽卡 (Perk) -> 挑战 BOSS -> 进入下一章。
  • 热量系统 (Heat System)：
    • 坦克的射击受到“热量”限制。
    • 普通子弹、火球、闪电链会产生不同程度的热量。
    • 当热量达到上限时，坦克将过热无法射击，必须等待冷却。
    • 策略点：玩家需要在输出火力和控制热量之间寻找平衡。
  • 自动回血：坦克具备自我修复功能，在不受伤害一段时间后会缓慢恢复生命值。

🎮 操作指南

游戏通过按键进行操作：

  • 移动控制：
    • Key 10：向上移动 (UP) / 菜单上选
    • Key 11：向下移动 (DOWN) / 菜单下选
    • Key 15：向左移动 (LEFT)
    • Key 7：向右移动 (RIGHT)
  • 功能控制：
    • Key 3：确认选择 (Select) —— 用于在升级界面选择 PerkCard。
  • 攻击：
    • 坦克会自动锁定并攻击，自动判定冷却和热量。

🃏 PerkCard (天赋卡) 系统详解

PerkCard 是游戏的核心 Roguelike 元素。每当玩家升级（特定等级）时，系统会随机提供 3 张卡片供玩家选择。

以下是所有卡片的效果说明：

🛡️ 生存与恢复类

  1. HEAL_SPEED_UP (回复速度提升)
    • 效果：加快自动回血的频率。让你在受损后能更快开始修复。
  2. HEAL_AMOUNT_UP (回复量提升)
    • 效果：增加每次回血恢复的生命值数值。
  3. HEALTH_UP (生命上限提升)
    • 效果：大幅提升坦克的最大生命值，并且瞬间回满所有生命值。救命神卡。

⚔️ 攻击与火力类

  1. ATTACK_UP (攻击力提升)
    • 效果：直接增加所有子弹的基础伤害。
  2. ATTACK_SPEED_UP (攻速提升)
    • 效果：减少射击的冷却时间，让你能更频繁地发射子弹（需配合热量卡使用，否则容易过热）。
  3. UNLOCK_FIREBALL (解锁：火球弹)
    • 效果：解锁副武器火球。火球击中敌人后会造成范围爆炸伤害（AOE），适合清理成群的小怪。
  4. UNLOCK_LIGHTNING (解锁：闪电链)
    • 效果：解锁副武器闪电链。闪电具有极高的单体伤害倍率，是击杀 BOSS 的利器。
  5. FIREBALL_RANGE_UP (火球范围提升)
    • 效果：扩大火球爆炸的伤害半径，增强 AOE 能力。
  6. LIGHTNING_MULTIPLIER_UP (闪电伤害倍率提升)
    • 效果：大幅提升闪电链的伤害倍数，使其成为终极杀伤武器。

🔥 热量管理类 (核心机制)

  1. HEAT_CAPACITY_UP (热容上限提升)
    • 效果：提升热量槽上限。允许你在过热前连续发射更多子弹。
  2. HEAT_COOL_DOWN_UP (散热速度提升)
    • 效果：加快热量的自然冷却速度。让你在停止射击后能更快恢复开火能力。

🚀 机动类

  1. MOVE_SPEED_UP (移速提升)
    • 效果：提升坦克的移动速度，更利于躲避密集的弹幕和敌人的冲撞。

👾 敌人图鉴 (山海经机械兽)

你将面对形态各异的敌人，它们拥有不同的攻击模式：

普通敌人

  • 飞廉 (Feilian)：小型敌人。移动速度极快，轨迹飘忽不定，通常成群结队出现进行骚扰。
  • 蛊雕 (Gudiao)：中型敌人。擅长远程伏击，从身体两侧发射高伤子弹，死亡时会发射一颗复仇火球。
  • 魑魅 (ChiMei)：极小敌人。只会进行自杀式冲撞，速度极快，一旦接触造成大量伤害。

关卡 BOSS

  • 饕餮 (Taotie)：贪婪巨兽，四凶之一。血量极厚，会使用吞噬技能将玩家吸向自己，或发起猛烈的冲撞和碾压。
  • 梼杌 (Taowu)：凶暴巨兽，四凶之一。擅长闪现移动，能释放覆盖全屏的随机弹幕和诡雷。
  • 相柳 (Xiangliu)：灾难巨兽。九头蛇身，能同时发射九排弹幕，释放闪电与火球，并能召唤大量小怪协同作战。

最后，祝您在 Chip Tanks 的世界中武运昌隆
